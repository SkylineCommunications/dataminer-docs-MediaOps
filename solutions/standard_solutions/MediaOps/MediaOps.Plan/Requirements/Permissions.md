# Minimum Required User Permissions

A user interacting with MediaOps Plan should at least have access to the following in DataMiner.

## Rights

- General > DataMiner web apps
- General > Elements > Access
- General > Elements > Data Display > Access
- Modules > Automation > Execute
- Modules > Bookings
- Modules > Profiles > UI available
- Modules > Profiles > All except instances
- Modules > Resources
- Modules > Scheduler
- Modules > Services > UI available
- Modules > User-definable apps > View apps

## Views

- Root View > DataMiner Catalog > MediaOps Plan View
- Root View > DataMiner Catalog > People and Organizations
